# DeepInsight Analyst: AI 驱动的买方投研框架

`DeepInsight Analyst` 是一个基于 CAMEL Agent 框架构建的自主 AI 股票分析系统。

本项目旨在解决一个核心问题：**如何从海量的、非结构化的市场信息（如研报、财报、新闻）中，提炼出真正具有深度和投资价值的“洞见” (Insight)，而不仅仅是“信息” (Information)？**

本项目的核心目标**不是**简单地总结信息，而是模拟一个专业的**买方（Buy-side）研究团队**的工作流。它能将一个模糊的分析请求（例如“分析贵州茅台”）转化为一份结构化的、由**论点驱动 (Thesis-Driven)** 的深度研究报告。

它强调一个批判性的工作流：**首先建立一套可证伪的分析框架（由Planner完成），然后通过深度研究（由Analyst执行）来发现并验证一个核心投资论点（The Thesis）**。最终产出的投资建议是基于严谨的交叉验证，而非简单的信息堆砌。

## 核心架构：双代理“投研团队”

本系统通过两个高度协同的 AI 代理，模拟“研究总监”与“分析师”的专业分工：

### 1. 🧠 总控规划师 (Master Planner)

* **角色：** 经验丰富的买方研究总监。
* **任务：** (由 `planner_prompt` 定义) 接收用户的初始查询，并将其分解为一份逻辑严谨、高层级的**“作战大纲” (High-Level Outline)**。
* **特点：**
    > 这个计划是**论点驱动**的，强调“质性优先、量化验证”。它会优先安排对**商业模式**和**护城河**的分析，然后再利用财务数据去**验证**这些质性判断。

### 2. 👨‍💻 股票分析师 (Stock Analyst)

* **角色：** 负责执行的一流分析师。
* **任务：** (由 `s_p_template` 定义) **严格按照**“总控规划师”制定的计划，一步一步地执行，并负责进行**深度研究 (Deep Research)**，尤其是从研报等非结构化数据中提取和验证信息。
* **工作流：**
    > 这是一个“**思考 -> 调用工具 -> 综合洞见**”的循环。分析师必须在每一步中思考需要什么信息，然后从工具箱（`ak_tools`, `read_url_tool` 等）中选择**一个**工具来获取数据，并对结果进行批判性分析。

### 示例一：贵州茅台 (600519)

#### 1. 用户的模糊请求

> “请帮我分析A股股票600519”

#### 2. “总控规划师”生成的作战大纲 (The Plan)

1.  拉取600519（贵州茅台）最新年报、三季报及Wind/同花顺一致预期，速览收入、净利润、毛利率、净利率、ROE、库存周转、批发/直销占比、i茅台占比、渠道库存周数、批价（整件/散瓶）及P/E、EV/EBITDA、PEG、相对沪深300溢价。
2.  检索近3个月卖方深度、行业月报、公司调研纪要、经销商交流纪要、糖酒会专家访谈，提炼市场对2025-26年盈利增速、批价走势、渠道库存、政策风险的一致预期与分歧点。
3.  质性拆解商业模式：高端茅台酒（非标、飞天）的社交货币属性→“价”比“量”更重要；系列酒（1935等）的放量逻辑；直销（i茅台+自营店）提升吨价与盈利；预收款+配额制带来的现金流护城河。
4.  护城河验证：①品牌：茅台在超高端白酒市占率>60%，价格带领先五粮液/老窖幅度；②转换成本：宴请/送礼场景下的品牌黏性；③供给限制：核心产区赤水河产能天花板（环境承载+政府配额）；④财务映射：毛利率>90%、净利率>50%、ROE>30%、经营现金流/净利润>1、预收款/收入≈30%。
5.  行业格局：用波特五力评估——高端赛道竞争者（五粮液、老窖、郎酒、习酒）产能扩张计划；新进入者（酱酒新势力）对次高端价格带冲击；地方政府对赤水河环保/产能审批趋严；消费者代际切换（年轻群体渗透率）；供应商议价（高粱/包材成本占比<10%可忽略）。
6.  宏观与政策：CPI温和通胀对高端酒价中性；政府反腐/限高消费政策尾部风险；地方政府债务压力→商务宴请预算弹性；人民币兑美元汇率对海外经销商利润影响；潜在消费税改革（从价+从量）情景测算。
7.  财务深潜：①将报表拆分为“茅台酒/系列酒/其他”三大分部，测算2025-26年可投放量（基酒储量→4年后可销系数）、吨价提升幅度（直销占比提升+非标占比+提价假设）；②构建“价/量/税率”三因子模型，验证利润弹性；③压力测试：若2026批价跌10%，对收入/净利润影响；④负债表：无有息负债、货币资金/总资产>50%，现金价值如何计入估值。
8.  估值与情景：①DCF：显性2025-29E FCFF→永续3% g，WACC 7.5-8%，对批价复合增速、直销占比、税率做Tornado分析；②相对估值：对标全球烈酒龙头（Diageo、Pernod）P/E vs. 增速，解释中国溢价；③得出合理市值区间，对比当前价格，指出市场预期隐含批价/销量假设。
9.  形成核心投资论点：例如“市场低估茅台在直销渠道持续提价+非标放量带来的吨价弹性，隐含2026批价复合增速仅3%，低于历史中枢7%，存在XX元/瓶上调空间→盈利上调15%→DCF重估+20%。”
10. 催化剂：①2025Q4-2026Q1飞天出厂价潜在上调；②i茅台上线新品（散花飞天、100ml小酒）放量；③年度股东会释放2026基酒产能指引；④国企改革/特别分红/回购信号。
11. 关键风险：①政策严厉限制“三公”消费或高端酒宴请；②经济超预期下行导致批价跌破2000元心理关口→渠道去库存负反馈；③赤水河环保政策导致产能缩减超预期，引发销量天花板提前；④估值回撤风险：无风险利率上行→消费股风险溢价扩大。

> ### 一、关键结论速览
>
> 1.  **估值：** 市场把小米当“纯硬件公司”定价——2025E P/E≈8–9 倍，显著低于三星（12×）、苹果（18×），也未计入互联网高毛利业务与汽车销量期权。

> 3.  **利润结构：** 互联网服务收入仅占 11%，却贡献约 60% 经营利润，毛利率>70%[1]；硬件（手机+IoT）收入占 89%，毛利率仅 9–14%，形成“低毛利硬件获客 + 高毛利互联网变现”的飞轮。
> 4.  **催化剂可见：** SU7 月交付已破 1 万辆，订单峰值 1.2 万辆；若 2025 年月均稳态 2 万辆，汽车业务有望从亏损扭向盈亏平衡，触发 SOTP 折价收敛。
>
> ### 二、术语解释
>
> * **SOTP（Sum-of-the-Parts）估值：** 把不同业务分别用合适估值指标定价，再加总。小米四块业务适用不同倍数——手机用 P/E、IoT 用 EV/EBITDA、互联网用 P/E、汽车用 P/S。
> * **ROIC > WACC：** 投资资本回报率高于加权平均资本成本，才是价值创造。小米 2023 ROIC≈8–9%，略高于 WACC（7.5%），处于价值创造临界点。
> * **可转债摊薄：** 2026–2027 年到期三批可转债（合计约 12 亿美元），转股价 HK$24–36；若股价持续低于 HK$20，到期可能以现金赎回，消耗净现金。
>
